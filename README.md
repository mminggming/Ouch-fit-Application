## OuchFit – Smart Outfit Planning with Interactive Styling & Intelligent Insights

OuchFit is a mobile application designed to help users organize, plan, and enhance their daily outfit choices through an interactive and structured experience. The application combines wardrobe management, calendar-based planning, and intelligent system design to simplify outfit decisions and improve usability.

### Key Features

- Outfit Planning and Calendar Integration  
  Users can schedule outfits in advance and visualize them through a calendar-based interface.

- Interactive Wardrobe Management  
  Manage clothing items digitally and combine them into outfits with an intuitive interface.

- Drag and Drop Styling  
  Create and customize outfits visually using flexible interactions.

- Multimedia Integration  
  Supports image selection, manipulation, and additional input methods to enhance user experience.

- Authentication and User Management  
  Integrated Google Sign-In and Firebase services for user authentication and scalability.

- Dynamic API Configuration  
  Environment-aware API system with automatic detection, fallback, and caching for seamless development across devices.

### Intelligent Insights (System Design)

The application is structured to support intelligent recommendations based on user behavior and interaction patterns.

- Analyze outfit usage trends and frequently selected items  
- Identify underutilized clothing  
- Enable future context-aware recommendations  
- Designed to evolve into a personalized styling assistant  

### Tech Stack

- React Native (Expo)
- TypeScript
- Expo Router
- Firebase
- AsyncStorage
- Axios
- Express
- Expo Media APIs

### Technical Highlights

- Modular and scalable architecture separating UI, logic, and configuration  
- Robust API handling with dynamic environment detection  
- Integration of native features including authentication and media handling  
- Secure configuration practices by excluding sensitive data from version control
